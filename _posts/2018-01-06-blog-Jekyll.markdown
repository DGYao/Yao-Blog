### [官网 →](https://www.jekyll.com.cn/)

### window下操作

- [下载Ruby+Devkit →](http://rubyinstaller.org/downloads/)

- 安装
```scss
	gem install jekyll
	gem install jekyll-paginate
```
		
- 切换到你的工程路径下
```scss
	jekyll serve --watch 
```

- 访问本地项目[localhost:4000 →](localhost:4000)
